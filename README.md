# 🧬 Multiple Disease Prediction System

A Machine Learning powered web application that predicts multiple diseases with high accuracy. This project is developed as part of my **MSc Computer Science Final Year Project (2026)**.

## 🚀 Live Demo
[Yahan deployment ke baad Streamlit ka link daal dena]

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **ML Libraries:** Scikit-learn, Pandas, NumPy
- **Models Used:** Random Forest, SVM, Extra Trees

## 📋 Features
- **Diabetes Prediction:** Based on 8 clinical parameters.
- **Heart Disease Prediction:** Based on 9 clinical parameters.
- **Kidney Disease Prediction:** Based on 15 clinical parameters.
- **Real-time History:** Tracks all predictions made during the session.
- **Interactive UI:** Professional medical-themed interface with reference ranges.

## 💻 How to Run Locally
**1. Clone the repository:**
   git clone [https://github.com/Aapka-Username/Multiple-Disease-Prediction-System.git](https://github.com/Aapka-Username/Multiple-Disease-Prediction-System.git)

**2. Install requirements:**
  pip install -r requirements.txt

**3. Run the app:**
  streamlit run app.py
